# 競品分析 + 爆款內容分析完整指南

---

## Part 1：競品分析框架

### 1.1 有效市場悖論（Booster 20220707）——核心前提

> **方法流通 → 差異消失 → 有效性消失**

當一個行銷方法被廣泛抄襲，它就失效了。

**應用**：
- 分析競品在用什麼方法 → 評估這些方法還有多久會被稀釋
- 你的差異化要建立在競品還沒佔領的位置
- 差異化 vs 成本領先不能同時追求（同時追求 = 最高倒閉率）

### 1.2 貝特森四維度分析（Booster 22）

| 維度 | 分析問題 |
|---|---|
| **空間涵蓋度** | 競品覆蓋了哪些客群？有哪些客群沒有覆蓋？ |
| **時間有效性** | 競品的差異還有多久會被模仿、被消滅？ |
| **轉換效能** | 競品把潛在客戶轉化成付費客戶的速度和品質如何？ |
| **整體世界觀** | 競品用什麼價值觀設計？這個世界觀能持續多久？ |

### 1.3 競品內容分析 SOP

```
Step 1：列出競品最高觀看的 5-10 支影片
Step 2：分析每支影片的 Hook 類型
        （封閉性？顛覆常識？情境代入？數字型？）
Step 3：分析觸及的是 L1/L2/L3 哪個層次
Step 4：分析他們的 CTA 類型（雙重束縛？單一選項？）
Step 5：評估差異被複製的風險（有效市場悖論）
Step 6：找出競品沒有覆蓋的受眾空白或痛點空白
Step 7：識別競品的世界觀（他們是「網紅型」還是「系統型」？）
```

### 1.4 競品世界觀分類（Booster 22）

| 類型 | 特徵 | 護城河 | 風險 |
|---|---|---|---|
| **網紅路線** | 靠個人魅力、話題、娛樂性 | 沒有，容易被複製 | 流量大但轉化低，品牌依附個人 |
| **系統路線** | 靠課程體系、學員成果、方法論 | 有，難被複製 | 起步慢，需要時間積累 |

你的差異化建議：建立「系統路線」的護城河，用「網紅路線」的內容方式傳播。

## Part 2：爆款內容分析框架

### 2.1 爆款的本質

爆款不是「好看的內容」，是「觸發框架轉變的內容」。

**爆款三要素**
| 要素 | 說明 | 如果缺少 |
|---|---|---|
| **差異**（觀眾停下來）| 違反預期、顛覆常識、讓人意外 | 觀眾滑過 |
| **共鳴**（觀眾留下來）| 說出了觀眾沒說出口的感受（L2/L3）| 觀眾看完就走 |
| **行動理由**（觀眾分享）| 讓人覺得「我要讓朋友也看到這個」| 留看不分享 |

### 2.2 爆款分析問卷（每支影片回答）

```
基本數據：
- 觀看數 / 完播率 / 分享數 / 留言數 / 互動率

內容分析：
1. 它違反了什麼「常識」或「預期」？（差異來源）
2. 觸及的是 L1/L2/L3 哪個層次痛點？
3. Hook 類型是哪種？（顛覆/故事/數字/身份/情境）
4. 它提供了什麼「框架」讓觀眾用新方式看舊問題？
5. 觀眾看完後的內部狀態？（頓悟？憤怒？感動？想試試？）

留言分析：
6. 留言裡最高頻的詞是什麼？（= 觀眾真正的 L2/L3 痛點）
7. 有沒有「他在說我！」這類留言？（= 冰山文案命中）
8. 有沒有「為什麼我之前不知道？」這類留言？（= L2 框架轉換成功）

可複製性：
9. 這個爆款的差異點是否可複製？還是一次性？
10. 有效市場悖論評估：這個方法多快會被競品抄走？
```

### 2.3 留言分析——提取真實痛點

留言比任何市場調查都準確——那是觀眾主動說出來的。

**分析方法**
```
Step 1：爬取高讚留言（按讚數前 20%）
Step 2：分類：
        A. 「他在說我！」型 → L3 自我認同觸點（最強付費動機）
        B. 「我也想知道⋯⋯」型 → 未滿足需求（下支影片主題）
        C. 「為什麼⋯⋯」型 → L2 問題（框架轉換機會）
        D. 負面質疑型 → 了解抗拒來源（下一個要克服的障礙）
Step 3：提取高頻詞 → 這些詞就是觀眾的語言，直接用在 Hook 裡
Step 4：把 A 類留言的語言直接放進下支影片的 Hook
```

**留言語言 = 最精準的 Hook 素材**

觀眾自己說出來的話，比你想的任何 Hook 都更精準。

## Part 3：數據驅動的差異化策略

### 3.1 差異評估矩陣

分析你和競品的差異，找到你的「可防禦的差異點」：

```
強差異（競品沒有）+ 觀眾需要 = 核心競爭優勢
強差異（競品沒有）+ 觀眾不知道需要 = 需要教育市場
弱差異（競品也有）+ 觀眾需要 = 紅海競爭
弱差異 + 觀眾不需要 = 無效投入
```

### 3.2 差異化組合策略

差異化來源（可以疊加）：
1. **方法論差異**：「我用神經科學解釋唱歌」vs「一般技巧教學」
2. **受眾差異**：「專攻上班族」vs「所有人」
3. **呈現方式差異**：「科學實驗式展示」vs「一般教學」
4. **結果差異**：「第一堂課保證感覺到改變」vs「長期訓練」
5. **故事差異**：「老師自己的突破故事」（無法被複製）
