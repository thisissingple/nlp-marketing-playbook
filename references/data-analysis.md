# 行銷數據分析完整指南
> 影片 Performance / 留言分析 / SEO 數據 / 廣告數據 / 煙圈閉環

---

## 1. 核心框架——煙圈數據分析法（Booster 17/21）

```
你的假設（左）→ 市場反應（右）→ 更新假設（左）→ 循環優化
```

每一個數據都是回饋。數據不是終點，是更新地圖的原料。

---

## 2. 影片 Performance 分析框架

### 2.1 關鍵指標診斷表

| 指標異常 | NLP 診斷問題 | 對應優化 |
|---|---|---|
| **完播率 < 30%** | Hook 不夠強？前 3 秒沒差異？ | 換 Hook 類型，加強封閉性 |
| **完播率 30-60%** | 中段失速？L2 轉換不夠衝擊？ | 加強 L2 框架轉換的衝擊感 |
| **完播率 > 70%** | 內容夠好，但轉化？ | 檢查 CTA 是否有雙重束縛 |
| **留言率低** | 沒有引導留言互動 | 在結尾加入「容易回答的問題」|
| **分享率低** | 沒有「讓別人也看到」的理由 | 加強 L2/L3 共鳴，加入「你的朋友也有這個問題嗎？」|
| **點擊率低（廣告）**| Hook 沒打到目標受眾的痛點 | A/B 測試不同 Hook 類型 |
| **轉換率低（廣告）** | CTA 有「不行動」選項，或沒有損失框架 | 改雙重束縛 CTA + 加損失框架 |

### 2.2 影片表現分層分析

```
頂部（完播率高 + 高互動）：
→ 分析：Hook 類型？觸及哪個層次痛點？框架是什麼？
→ 行動：複製這個公式，做 3-5 支變體

中部（完播率中等）：
→ 分析：哪個時間點掉人？（前 3s / 中段 / 結尾）
→ 行動：找到失速點，針對性優化

底部（完播率低 + 低互動）：
→ 分析：Hook 類型？是不是用了已流通的公式？
→ 行動：換公式，或換受眾定向
```

## 3. 留言分析——提取真實市場洞察

留言是最精準的市場調查。觀眾主動說出來的話，比任何問卷都準。

### 3.1 留言分類系統

```
A 類：「他在說我！」/ 「就是這樣！」
  → 意義：L3 自我認同命中，最強付費動機
  → 行動：把這些句子直接用在下支影片的 Hook

B 類：「為什麼⋯⋯」/ 「這個背後的原因是？」
  → 意義：觀眾想要 L2 框架轉換
  → 行動：做一支「為什麼」型的深度影片

C 類：「我也想學⋯⋯」/ 「要去哪裡找？」
  → 意義：有購買意願，需要 CTA
  → 行動：在留言回覆裡放 CTA，或做一支針對這個需求的影片

D 類：質疑型 / 負面型
  → 意義：抗拒點（未來的 L1 購買阻礙）
  → 行動：做一支直接回應這個質疑的影片（加強信任）

E 類：分享型 / 「這個要給⋯⋯看」
  → 意義：L2/L3 共鳴命中，可擴大傳播
  → 行動：分析是哪個觸點觸發分享，放大這個元素
```

### 3.2 留言頻率分析

```
Step 1：提取所有留言的高頻詞
Step 2：這些詞就是觀眾的語言，直接用在 Hook 裡
Step 3：高頻詞對應的情緒 = 觀眾真正的 L2/L3 痛點
Step 4：把這些痛點放進下個月的廣告素材
```

## 4. SEO 數據分析

### 4.1 關鍵指標

| 指標 | 正常範圍 | 異常診斷 |
|---|---|---|
| 點擊率（CTR）| 2-5%（一般），5%+（好）| 低 → 標題不夠吸引，換 Hook 類型 |
| 平均排名 | 1-10（目標）| 低 → 內容深度不夠，加強 L2 框架 |
| 停留時間 | 2-4 分鐘+ | 低 → 開場不夠吸引，或內容沒 L2 轉換 |
| 跳出率 | < 60% | 高 → 標題與內容不一致（期望落差）|

### 4.2 搜尋意圖匹配診斷

```
高排名但低點擊 → 標題不夠吸引（換 Hook 類型）
高點擊但高跳出 → 內容與標題不符（期望落差）
低排名但高停留 → 內容質量好，需要加強外部連結
```

## 5. 廣告數據分析

### 5.1 廣告漏斗診斷

```
曝光 → 點擊（CTR）→ 落地頁（停留）→ 轉換（CTA）→ 付費
                  ↑              ↑           ↑
              Hook 問題      內容問題    CTA 問題
```

### 5.2 A/B 測試框架

**每次只測試一個變數**：

| 測試項目 | 測試邏輯 |
|---|---|
| Hook 類型 | 顛覆型 vs 情境型 vs 數字型 |
| 痛點層次 | L1 vs L2 vs L3 |
| CTA 框架 | 獲得框架 vs 損失框架 |
| 受眾類型 | 依薩提爾四型分組測試 |

**測試週期**：至少 3-7 天，觀看數 > 1000 才能下結論。

## 6. 成交後設分析 SOP（Booster 5）

**目的**：找出成交客戶真正付錢的共通後設，植入廣告素材。

```
每月訪談 3-5 位成交客戶：
問 1：「在決定報名之前，你心裡在想什麼？」
問 2：「是什麼讓你最後決定的？」
問 3：「你當時最擔心什麼？（後來發現擔心是多餘的？）」

分析回答，提取：
A. 共通的 L2 生活影響描述 → 放進廣告素材
B. 那句「打動我的話」（L3 觸點）→ 直接用在 Hook
C. 最常見的購買前疑慮 → 做一支影片直接回應這個疑慮
```

## 7. 月度數據整合報告框架

```
1. 頂部影片分析（本月表現最好的 3 支）
   - Hook 類型 / 觸及層次 / 爆點在哪裡

2. 底部影片分析（本月表現最差的 3 支）
   - 哪裡失速 / 什麼公式已失效

3. 留言洞察
   - 本月高頻詞 / L3 觸點命中 / 質疑點

4. SEO 進展
   - 新排名 / 點擊率趨勢 / 搜尋意圖匹配度

5. 廣告數據
   - A/B 測試結果 / 最佳 Hook 類型 / 最佳 CTA

6. 下月假設更新
   - 基於回饋，更新哪些設計假設
   - 要驗證的新假設是什麼
```
