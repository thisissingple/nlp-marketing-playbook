# 廣告文案 / 行銷信件完整指南
> 薩提爾四型受眾 / 冰山文案 / 廣告結構 / 行銷信件模板

---

## 1. 薩提爾四型受眾分類（Booster 14）

識別受眾類型是文案的第一步。用 bio、留言、行為模式判斷。

| 類型 | 識別信號 | 文案語言策略 |
|---|---|---|
| **討好型** | 留言總是讚美別人、說「謝謝」、分享對他人有幫助的內容 | 強調「你值得這個」「這也會幫助你身邊的人」「不用委屈自己了」|
| **超理智型** | 留言問數據、問原理、問「為什麼有效」 | 給邏輯因果、研究佐證、系統化說明、不要只說「感覺」|
| **責備型** | 留言有點挑剔、強調自己的主見、不喜歡被說教 | 給選擇感「你可以決定」、強調「這是你的選擇」、不用「你應該」|
| **打岔型** | 留言很隨機、轉移話題、回應跳來跳去 | 輕鬆短句、幽默轉折、不要太嚴肅、製造意外感 |

**混合型處理**：大部分人是混合型，先辨認主要型，用該型語言開頭，後段混入次要型。

## 2. 冰山原則——說出觀眾沒說出口的（Booster 14）

```
觀眾說的（表層）：「我想改善唱歌」
觀眾感受（中層）：「每次唱歌都有壓力，怕被笑」
觀眾渴望（深層）：「我希望自己是那個讓全場震撼的人」← 文案要說這個
```

**範例**
- ❌ 「學習專業聲樂技巧，提升音準節奏。」
- ✅ 「不是每個人都能在舞台上讓全場安靜——但這件事是可以學的。」

## 3. 完整廣告文案結構

```
[Hook（封閉性操作）]
↓
[確認 L1 表層痛點——讓觀眾點頭]
↓
[放大 L2 生活影響——讓觀眾說「對，就是這樣」]
↓
[觸及 L3 自我認同——說出觀眾沒說出口的]
↓
[框架轉換——讓觀眾用你的視角看問題]
↓
[可證實差異——5 秒內可見的具體結果]
↓
[雙重束縛 CTA——沒有「不行動」的選項]
```

**可見差異原則**（Booster 22）：
- ❌ 「我們的教學更科學更系統。」
- ✅ 「學員在第一堂課後就能感覺到聲音改變——因為我們不是在教技巧，是在重新訓練神經系統。」

**損失框架 vs 獲得框架**（Booster 17）：

| 框架 | 效果 | 用法 |
|---|---|---|
| 獲得框架 | 適合已有渴望的受眾 | 「報名後你能⋯⋯」 |
| 損失框架 | 對尚未覺察的受眾更強 | 「每多等一週，你就繼續帶著這個問題⋯⋯」 |

一般損失框架比獲得框架強 2 倍。冷受眾（不認識你的）用損失，暖受眾（已追蹤你的）用獲得。

## 4. 行銷信件結構

### 開發信（Cold Email / DM）
```
主旨：[封閉性Hook，讓他必須打開]

開場：說出他的 L2 痛點（不說你自己）
中段：框架轉換（讓他用新視角看問題）
轉折：你的解法（1 句話，不要一次說完）
CTA：雙重束縛（兩個行動選項）

簽名：你是誰（1 句話）
```

### 暖受眾跟進信（已有互動）
```
開場：呼應上一次的互動（讓他感覺你記得他）
中段：L3 自我認同觸點（說出他的渴望）
解法：你能做什麼（具體，可證實）
CTA：低門檻行動（不要直接推銷）
```

### EDM / Newsletter
```
主旨：Hook 類型（顛覆 / 身份懸念 / 數字反差）
開場：L2/L3 痛點（3 句以內）
主體：L2 框架轉換 + 可見差異
結尾：單一 CTA（不要放多個）
```

## 5. 受眾細分文案策略

同一個產品，對不同受眾說不同的話：

| 受眾 | 主打痛點 | 公約數連結 |
|---|---|---|
| 學唱歌的人 | 突破音域、演唱技巧 | 唱歌×自信 |
| 想上台表演的人 | 舞台恐懼、怯場 | 唱歌×膽量×被看見 |
| KTV 愛好者 | 讓朋友刮目相看 | 唱歌×社交×影響力 |
| 職業需求者 | 說話有磁性、簡報有力 | 聲音×說服力×領導力 |
| 家長 | 孩子的音樂教育 | 唱歌×孩子自信×才藝 |
