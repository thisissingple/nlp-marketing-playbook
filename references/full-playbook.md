## MODULE 1：內容創作（影片 / 短影音 / 講稿）

### 1.1 Hook 設計——封閉性操作（Booster 2）

**原理**：訊息存在但刻意不說完 → 大腦無法閉合 → 強迫繼續看。

**公式**：`[暗示某事存在] + [不說完] + [讓觀眾腦補]`

**範例（歌唱教學）**
- ✅ 「大部分老師教你的方法，反而在傷害你的聲帶。」
- ✅ 「99% 的人唱不好，不是因為天分，是因為沒有人告訴他這件事。」
- ✅ 「我把照片給我的學員看，他說了一句話讓我現在還在想。」
- ❌ 「今天教你高音技巧。」（直接告訴，大腦不需要補）

**操作規則**
| 原則 | 說明 |
|---|---|
| 不說完 | 留下空缺讓觀眾腦補 |
| 暗示差異 | 「大部分人做錯」→ 觀眾想知道自己是不是那個人 |
| 不問句 | 問句讓觀眾可以說「不」就離開 |
| 30 字以內 | Hook 越短越好，不要解釋 |

---

### 1.2 內容深度——學習層次 L1→L2（Booster 9）

| 層次 | 定義 | 觀眾感受 |
|---|---|---|
| L0 | 資訊給了就用 | 「有用但明天忘了」 |
| L1 | 學到技巧能應用 | 「我學到東西了」 |
| **L2** | **框架轉變，世界觀改變** | **「我之前的想法是錯的！」** ← 目標 |

**操作**：先給常識（L1）→ 揭示背後更大框架（L2 轉換）

**範例**
- L1：「唱高音要放鬆喉嚨。」
- L2：「你喉嚨緊是因為大腦把高音跟危險連結——這是神經系統問題，不是技術問題。你需要重新訓練的不是喉嚨，是你對高音的內部反應。」

L2 內容讓人分享、讓人衝擊、讓人記住。

### 1.3 痛點設計——三層挖掘（Booster 5）

**三層結構**
```
L1 表層：「我唱歌唱不好」
L2 生活影響：「不敢在朋友面前唱，KTV 都要找藉口不去」
L3 自我認同威脅：「覺得自己沒天分，某些事就是不如別人」← 付費動機在這
```

**行銷應用**
- 廣告文案：先說 L1（觀眾認得）→ 描述 L2（他沒說出口的影響）→ 觸及 L3（自我認同）
- SEO 標題：用 L2/L3 作標題，點擊率比 L1 高

**公約數概念**：找產品跟多個痛點的共通元素，每增加一個連結就增加一類潛在客戶。
- 範例：唱歌 × 呼吸 = 可連結失眠、氣色、情緒調節、自信、冥想

### 1.4 CTA 設計——雙重束縛（Booster 5+17）

**原理**：讓對方沒有「不行動」的選項，怎麼選都對你有利。

**公式**：`[行動A] 或 [行動B]`（兩個都是行動）

**範例**
- ✅ 「你可以先預約體驗課，或先加 LINE 讓顧問了解你的狀況——哪個方便？」
- ✅ 「今天看完的人，直接點連結讓我們幫你評估聲音狀況。」
- ❌ 「有興趣可以報名體驗課。」（給了「不行動」的選項）

**損失框架（Booster 17）**：人對損失的恐懼 > 對獲得的期待。
- ❌ 「報名後你能學到高音技巧。」
- ✅ 「每多等一週，就是讓自己繼續帶著那個問題撐過去。」

## MODULE 2：廣告文案完整結構

```
[Hook（封閉性操作）]
→ [確認 L1 痛點]
→ [放大 L2 生活影響]
→ [觸及 L3 自我認同]
→ [框架轉換（讓他用你的視角看問題）]
→ [可證實差異（5 秒內可見）]
→ [雙重束縛 CTA]
```

**可見差異原則**（Booster 22）：不用抽象論述，用立即可感知的差異。
- ❌ 「我們的教學方法更科學、更系統化。」
- ✅ 「學員在第一堂課後就能感覺到聲音的改變——因為我們不是在教技巧，是在重新訓練神經系統反應。」

## MODULE 3：競品分析框架

### 3.1 四維度分析（Booster 22）

| 維度 | 分析問題 |
|---|---|
| 空間涵蓋度 | 競品覆蓋了哪些客群？有哪些沒有覆蓋？ |
| 時間有效性 | 競品的差異還有多久會被模仿、被消滅？ |
| 轉換效能 | 競品把潛在客戶轉化成付費客戶的速度和品質 |
| 整體世界觀 | 競品用什麼價值觀設計？這個世界觀能持續多久？ |

### 3.2 有效市場悖論（Booster 20220707）

> **方法流通 → 差異消失 → 有效性消失**

當一個行銷方法被廣泛抄襲，它就失效了。

**應用**：
1. 分析競品在用什麼方法 → 這些方法還有多久會被稀釋？
2. 你的差異化要建立在哪裡 → 不能跟競品打相同戰場
3. 差異化 vs 成本領先不能同時追求（同時追求 = 最高倒閉率）

### 3.3 競品內容分析 SOP

```
Step 1：列出競品最高觀看的 5-10 支影片
Step 2：分析 Hook 類型（封閉性？痛點？社會證明？）
Step 3：分析觸及的是 L1/L2/L3 哪個層次
Step 4：評估差異被複製的風險（有效市場悖論）
Step 5：找出競品沒有覆蓋的空白（你的機會）
```

## MODULE 4：爆款內容分析框架

### 4.1 爆款三要素

| 要素 | 說明 |
|---|---|
| 有差異（觀眾停下來） | 違反預期、顛覆常識、讓人意外 |
| 有共鳴（觀眾留下來） | 說出觀眾沒說出口的感受（L2/L3 痛點）|
| 有行動理由（觀眾分享）| 讓人覺得「我要讓朋友也看到這個」|

### 4.2 爆款分析問卷（每支影片回答）

```
1. 它違反了什麼「常識」或「預期」？（差異來源）
2. 觸及的是 L1/L2/L3 哪個層次痛點？
3. 觀眾看完後的內部狀態是什麼？（覺察升起？憤怒？感動？）
4. 它提供了什麼「框架」讓觀眾用新方式看舊問題？
5. 這個內容類型能持續產出嗎？還是一次性爆款？
```

## MODULE 5：行銷信件 / 文案

### 5.1 薩提爾四型受眾分類（Booster 14）

| 類型 | 識別特徵 | 文案語言 |
|---|---|---|
| 討好型 | 關注別人感受、怕麻煩別人 | 「這對你周圍的人也有幫助」「你值得有這個」 |
| 超理智型 | 相信數據/系統/規則 | 強調研究支撐、系統化、有邏輯的因果 |
| 責備型 | 強調主導權、不喜歡被說教 | 給選擇感「你可以決定」，不用「你應該」 |
| 打岔型 | 注意力分散、需要娛樂感 | 輕鬆有趣、短句、意外的轉折 |

### 5.2 冰山原則（Booster 14：薩提爾冰山模型）

好的文案說出觀眾內心在說但沒說出口的話。

```
觀眾說的（表層）：「我想改善唱歌」
觀眾感受（中層）：「每次唱歌都很有壓力，怕被笑」
觀眾渴望（深層）：「我希望自己是那個讓全場震撼的人」← 文案要說這個
```

**範例**
- ❌ 「學習專業的聲樂技巧。」
- ✅ 「不是每個人都能在舞台上讓全場安靜——但這件事是可以學的。」

## MODULE 6：SEO 內容策略

### 6.1 邏輯類型品牌（Booster 6）

貝特森說：類別（Class）比成員（Member）更有影響力。

```
差的 SEO：你是很多高音教學文章之一
好的 SEO：你是「科學化歌唱教學」這個類別的代表
```

**操作**：創造你專屬的術語和分類體系，讓競品變成你類別下的「成員」。

### 6.2 SEO 內容設計原則

| 原則 | NLP 對應 | 操作 |
|---|---|---|
| 建立獨特分類 | 邏輯類型（Booster 6）| 創造專屬術語和分類，讓你成為類別代表 |
| 觸及 L2/L3 痛點 | 學習層次（Booster 9）| 標題讓人有「我之前的認知可能是錯的」的感覺 |
| 可證實的差異 | 貝特森差異理論 | 每篇文章提供一個「立即可驗證」的洞見 |
| 內部連結生態 | 煙圈模型 | 文章互相引用，建立完整的知識煙圈 |

### 6.3 SEO 標題公式

```
類型 1（顛覆常識）：「為什麼你被教的[X]，其實讓你[負面結果]」
類型 2（框架轉換）：「你不是[表層問題]——你真正的問題是[更深層]」
類型 3（差異化）：「[你的獨特分類]：一種[競品類別]沒有的方法」
```

## MODULE 7：行銷數據分析

### 7.1 煙圈數據分析法

```
你的廣告假設（左）→ 市場反應（右）→ 更新假設（左）→ 循環
```

| 數據異常 | NLP 診斷問題 |
|---|---|
| 停留時間低 | Hook 不夠強？差異沒有觸發覺察？ |
| 點擊率低 | 訊息沒觸及 L2/L3 痛點？ |
| 轉換率低 | CTA 給了「不行動」選項？框架轉換不夠？ |
| 高分享率 | 確認哪個 L2/L3 觸點讓人想分享，放大它 |

### 7.2 成交後設分析 SOP（Booster 5）

**目的**：找出成交客戶真正付錢的共通後設，植入廣告素材。

```
每月訪談 3-5 位成交客戶：
問：「在決定報名之前，你心裡在想什麼？是什麼讓你最後決定的？」

分析回答，找出：
1. 共通的生活影響描述（L2 痛點）
2. 那句「打動我的話」（L3 自我認同觸點）
3. 把這些直接放進下個月的廣告文案
```
