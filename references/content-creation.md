# 內容創作完整指南
> Hook 設計 / 多元內容公式 / 短影音 / 講稿 / L1→L2 / 痛點 / CTA

## 1. Hook 設計——封閉性操作（Booster 2）

**原理**：訊息存在但刻意不說完 → 大腦無法閉合 → 強迫繼續。

**公式**：`[暗示某事存在] + [不說完] + [讓觀眾腦補]`

### Hook 類型庫（避免公式化，輪流使用）

**A. 顛覆常識型**
- 「大部分老師教你的方法，反而在傷害你的聲帶。」
- 「99% 的人唱不好，不是因為天分。」
- 「你以為在練習，但你其實在強化錯誤。」

**B. 身份懸念型**
- 「有一種人，開口的第一秒就讓全場安靜。他們做了一件你沒做的事。」
- 「你身邊那個唱歌很好聽的人，用的不是天分。」

**C. 數據反差型**
- 「我教了 300 個學員，其中 80% 的問題出在同一個地方。」
- 「花了 3 年練不好，換一個方法 3 週就突破。」

**D. 失落製造型**（Booster 21 推拉原理）
- 「我本來不打算說這件事……」
- 「這個方法我只告訴認真的人。」

**E. 情境還原型**（讓觀眾代入）
- 「KTV 輪到你，你把麥克風推給別人——這種感覺你懂嗎？」
- 「你練了很久，但上台還是怯場。你知道問題在哪裡嗎？」

**F. 社會反差型**
- 「為什麼有人不需要練習就能讓人心動？」
- 「同樣的歌，有人唱完大家喝酒，有人唱完全場鼓掌。差別是什麼？」

**操作規則**
| 原則 | 說明 |
|---|---|
| 不說完 | 留下空缺讓觀眾腦補 |
| 不問句 | 問句讓觀眾說「不」就離開 |
| 30 字以內 | Hook 越短越好 |
| 輪換類型 | 同一帳號不能用同一種 Hook 超過 3 次 |

## 2. 內容深度——學習層次 L1→L2（Booster 9）

| 層次 | 定義 | 觀眾感受 |
|---|---|---|
| L0 | 資訊給了就用 | 「有用但明天忘了」 |
| L1 | 技巧，能應用 | 「我學到東西了」 |
| **L2** | **框架轉變，世界觀改變** | **「我之前的想法是錯的！」← 目標** |

**L1→L2 轉換方法**

1. 先給觀眾熟悉的「常識」（L1）
2. 揭示這個常識背後的更大框架（L2 轉換）
3. 讓觀眾發現：「原來我之前理解的是片面的」

**範例**
- L1：「唱高音要放鬆喉嚨。」
- L2：「你喉嚨緊是因為大腦把高音跟危險連結——這是神經系統問題，不是喉嚨問題。你需要重新訓練的是大腦的內部反應，不是肌肉。」

**L2 觸發公式**
- 「你以為是 A，其實是 B。」
- 「大家都在解決表面問題，但真正的問題是⋯⋯」
- 「你沒有 X 的問題，你有的是 Y 的問題。」

## 3. 痛點設計——三層挖掘（Booster 5）

```
L1 表層：「我唱歌唱不好」
L2 生活影響：「不敢在朋友面前唱，KTV 找藉口不去」
L3 自我認同威脅：「覺得自己沒天分，某些事就是不如別人」← 付費動機
```

**公約數概念**：找產品與多個痛點的共通元素，每增加一個連結就增加一類潛在客戶。
- 唱歌 × 呼吸 = 可連結失眠、氣色、情緒調節、自信、冥想、專注力

## 4. CTA 設計——雙重束縛（Booster 5+17）

**公式**：`[行動A] 或 [行動B]`（兩個都是行動）

**損失框架**（更強）：
- ❌ 「報名後你能學到高音技巧。」
- ✅ 「每多等一週，就是讓自己繼續帶著那個問題撐過去。」

## 5. 多元內容公式——差異排列組合

**重要原則**：同一帳號的內容不能用同一種公式超過 3 次。
差異消失 = 訊息消失 = 觀眾麻木。必須持續輪換。

### 公式 A：顛覆型（違反常識）
```
Hook（顛覆）→ 說明「常識為什麼錯」→ 提供正確框架 → 行動呼籲
```
適合：漲粉、擴大觸及、吸引不熟悉的受眾

### 公式 B：故事型（情境代入）
```
Hook（情境）→ 角色痛點故事 → 轉折點（方法）→ 結果 → CTA
```
適合：讓人有「他在說我」的感覺，提升共鳴

### 公式 C：框架型（給新視角）
```
Hook（問題）→ 揭示更大框架（L2）→ 新視角下的解法 → CTA
```
適合：建立專家權威、培養信任

### 公式 D：對比型（可見差異）
```
Hook → 錯誤做法（很多人在做）→ 正確做法（你的方法）→ 即時可見差異 → CTA
```
適合：直接展示差異，適合已知受眾深化信任

### 公式 E：數字型（量化說服）
```
Hook（數字）→ 數字背後的故事 → 如何複製這個結果 → CTA
```
例：「300 個學員中，80% 的問題出在這裡」

### 公式 F：問答型（直接回應搜尋意圖）
```
問題（觀眾常問的）→ 直接回答 → 深化（為什麼）→ CTA
```
適合：SEO、知識型內容

### 公式 G：情緒引爆型（直接打 L3）
```
Hook（L3 自我認同）→ 共鳴（你不是一個人）→ 解法 → CTA
```
例：「唱歌唱不好不是你的問題——是沒有人教你真正的原因。」

**輪換建議**
- 一週 7 支影片：A×2、B×1、C×1、D×1、E×1、G×1
- 不同公式之間不重複 Hook 類型
- 每個月分析哪個公式表現最好，下個月加重該公式但其他仍要保留

## 6. 短影音結構模板

### 結構 1：15 秒版
```
0-3s：Hook（封閉性 / 顛覆）
3-10s：核心洞見（L2 框架一句話）
10-15s：CTA（雙重束縛）
```

### 結構 2：30-60 秒版
```
0-3s：Hook
3-15s：問題放大（L2 痛點）
15-45s：解法 / 框架
45-60s：CTA
```

### 結構 3：1-3 分鐘版（深度型）
```
0-5s：Hook
5-30s：故事 / 情境代入
30-90s：問題本質（L2 轉換）
90-150s：解法展示
150-180s：CTA + 留言互動引導
```

## 7. 留言互動設計

留言是「右邊回饋」，用貝特森煙圈閉環：
留言分析 → 更新內容假設 → 下支影片

**引導留言的方式**
- 在影片結尾問一個 L1 層次的問題（容易回答）：「你有這個問題嗎？留言告訴我」
- 不要問太深的問題——觀眾懶得回答
- 留言裡的高頻詞 = 觀眾真正的 L2/L3 痛點，直接拿來做下支影片的 Hook
