## 3. Singple 的受眾描述

### 主力受眾 A：「高音困擾者」
- **年齡**：25-45 歲（推測，成人）
- **L1**：高音上不去、喉嚨緊
- **L2**：KTV 不敢唱、不敢在人前表現
- **L3**：「我可能就是沒有音樂天分的那種人」
- **行銷重點**：顛覆「天分論」，提供科學化解釋（神經系統、肌肉記憶）

### 主力受眾 B：「舒壓/興趣學習者」
- **L1**：想學唱歌作為興趣
- **L2**：平常壓力大，想要一個出口
- **L3**：想要一個「只屬於自己的成就」
- **行銷重點**：連結唱歌 × 情緒出口 × 自我價值

### 主力受眾 C：「偶像崇拜者」
- **L1**：想唱偶像的歌
- **L2**：和偶像的距離感（他能做到我做不到）
- **L3**：「如果我也能像偶像一樣唱⋯⋯」
- **行銷重點**：「偶像也是這樣練出來的」的框架

## 5. 行銷 Agent 的決策優先級

### 當任務是「產出內容」時：

```
Step 1：判斷目標受眾是 A/B/C 哪一類
Step 2：選對應的 L3 觸點
Step 3：選內容公式（從 content-creation.md）
Step 4：確保 Hook 不重複近期公式
```

### 當任務是「分析數據」時：

```
Step 1：查 daily-cache 取得最新數據
Step 2：比對目標值（月目標 170 萬 TWD）
Step 3：找出漏斗哪一環掉了最多
Step 4：用 NLP 診斷問題（data-analysis.md）
Step 5：輸出 3 條優先行動建議
```

### 當任務是「競品分析」時：

```
Step 1：找出競品的 Hook 類型
Step 2：判斷競品打的是 L1/L2/L3 哪層
Step 3：找出競品沒有覆蓋的空白
Step 4：對比有效市場悖論——這個方法還有多久失效？
Step 5：建議 Singple 的差異化方向
```

## 7. 競品對比（從爆款偵測推斷）

Singple 的主要競品類型：
- **傳統音樂教室**：強調老師資歷、教學環境（打 L1）
- **網路課程平台**（PressPlay 等）：強調性價比、隨時學（打 L1）
- **YouTube 免費內容創作者**：強調顛覆常識、病毒式傳播（打 L2）

**Singple 的護城河**：科學化系統 + 高轉正率 + 一對一高端路線——這些是競品難以快速複製的。

**要防守的點**：內容差異化（有效市場悖論）——方法被抄走之前，持續用學員真實案例和老師獨特性建立壁壘。
