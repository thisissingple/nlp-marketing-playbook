# NLP 行銷 Playbook

蒸餾自貝特森（Gregory Bateson）+ 薩提爾（Virginia Satir）課程知識庫的行銷應用框架。

## 適用場景

- 影片 / 短影音 / 講稿產出
- 廣告文案 / 行銷信件
- 競品分析
- 爆款內容分析
- SEO 內容策略
- 行銷數據分析與優化建議

## 核心原則

> 行銷的本質：讓對方的內部活動自己生成渴望，讓你成為滿足那個渴望的唯一選項。

## 使用方式

### Agent 安裝（OpenClaw Skill）

1. 下載 `nlp-marketing-playbook.skill`
2. 放進 agent 的 skills 目錄
3. 或直接克隆此 repo，指向 skill 目錄

### 直接閱讀

- `SKILL.md` — 快速參考（10大原則）
- `references/full-playbook.md` — 完整 7 Module 內容
